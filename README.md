# 🎓 SMS-School-ERP

A comprehensive School ERP (Enterprise Resource Planning) system built using **Django**. This application helps schools manage students, teachers, admissions, authentication, and other academic activities through a modern web interface.

---

## 🚀 Features

- 👨‍🎓 Student Management
- 👩‍🏫 Teacher Management
- 🔐 Secure User Authentication
- 📝 Admission Management
- 📄 Academic Information
- 📞 Contact & Enquiry Forms
- 📂 Dashboard for Different Users
- 📱 Responsive User Interface
- 📊 School Information Pages

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/anjaliagrahari010/SMS-School-ERP.git
```

### Move into the project

```bash
cd SMS-School-ERP
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

Windows

```bash
venv\Scripts\activate
```

Linux/Mac

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Apply Migrations

```bash
python manage.py migrate
```

### Run the Server

```bash
python manage.py runserver
```

Visit:

```
http://127.0.0.1:8000/
```
